
# MUSIC PLAYER

This is a simple music player software made with python programme.


## Run Locally

Clone the project

```bash
  git clone https://github.com/muzammil-85/MUSICPLAYER.git
```

Go to the project directory

```bash
  cd MUSICPLAYER
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Start the server

```bash
  python main.py
```
